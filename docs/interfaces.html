TypeScript Interfaces

Complete reference for all TypeScript interfaces used in SweetMeSoft JS library.

OptionsSelect

Basic options to generate a Select. This method uses Bootstrap Select.

Reference: Bootstrap Select Documentation

Properties:
url mandatory
string

URL to get the data

data optional
string | Object | FormData
default: {}

Parameters to send to the URL

dropDowns mandatory
JQuery[]

jQuery objects where the select will be generated

Example: [$('#select')]

autoSelect optional
boolean
default: true

If true and there is only one option, it will be selected automatically

callback optional
(list: any) => void

Callback called when the options are loaded in the selects

enable optional
boolean
default: true

If true and the select is disabled, it will be enabled automatically

text optional
string
default: 'name'

Object property to show in the main text

internal optional
string
default: 'id'

Object property to add in the value for each option

jwt optional
string
default: ''

JWT token for authentication

lang optional
string
default: 'en-US'

Language/culture code for localization

OptionsRequest

Configuration options for AJAX requests with authentication and localization support.

Properties:
url mandatory
string

The endpoint URL for the request

data optional
string | Object

Data to send with the request

successCallback optional
(response: any) => void

Function to call on successful response

errorCallback optional
(response: any) => void

Function to call on error response

jwt optional
string
default: ''

JWT token for authentication

lang optional
string
default: 'en-US'

Language/culture code for localization

showSuccess optional
boolean
default: true

Whether to show success messages

showError optional
boolean
default: true

Whether to show error messages

OptionsChart

Configuration options for generating charts using Chart.js.

Properties:
chart mandatory
JQuery

Canvas element where the chart will be painted

url mandatory
string

Endpoint where the chart data will be fetched

data optional
string | Object
default: null

Parameters for the GET request

interval optional
'daily' | 'monthly'

Chart interval type - daily shows 1-31 on X axis, monthly shows Jan-Dec

subFilter optional
string

Field name of data response used to split info in same interval

OptionsMap

Configuration options for Google Maps integration.

Properties:
edtLatitude mandatory
JQuery

Input element for latitude value

edtLongitude mandatory
JQuery

Input element for longitude value

showCurrentLocation optional
boolean
default: true

Show the current location in the map

modal optional
boolean
default: true

If the map should be displayed in a modal

showAutocomplete optional
boolean
default: false

Show an autocomplete field above the map

showCoordinates optional
boolean
default: false

Show a label with latitude and longitude at bottom of the map

OptionsModal

Basic options to generate a modal using Bootstrap Modal.

Reference: Bootstrap Modal Documentation

Properties:
title mandatory
string

Title of the modal

type mandatory
'view' | 'html'

Where the modal will get the render view

view: load from service URL | html: load static HTML

loadCallback optional
Function
default: undefined

Event fired when the view is loaded in DOM

primaryCallback optional
() => Promise<boolean>
default: undefined

Event fired when the primary button is clicked

size optional
'small' | 'big'
default: 'small'

Width size for modal. Small is default bootstrap width, big is 80% screen width

OptionsCropper

Basic options to generate a square cropper using Cropper.js loaded in a SweetAlert2 modal.

References: Cropper.js | SweetAlert2

Properties:
uploadControl mandatory
JQuery

jQuery object where the user will upload the image

imgControl mandatory
JQuery

jQuery object where the cropped image will be painted

aspectRatio optional
'square' | '2/3' | '3/2' | '4/3' | '3/4' | '16/9' | '9/16' | '20/9' | '9/20'
default: 'square'

Aspect ratio for the cropper

callback optional
(blob: any) => void

Callback function executed when the image is cropped

OptionsTable

Basic options to generate a table using DataTables.

Reference: DataTables Documentation

Key Properties:
table mandatory
JQuery

Table element to generate DataTable

type optional
'url' | 'list'

Way to get information - from URL or from provided list

url optional
string

URL to get the data (mandatory if type is 'url')

data optional
any[]

List of objects to create the table (mandatory if type is 'list')

rowsPerPage optional
number
default: 10

Number of rows per page

canOrder optional
boolean
default: true

Enable or disable ordering in table
