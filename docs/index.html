Ajax Methods

The Ajax module provides simplified HTTP request methods with automatic error handling, JWT authentication, and localization support.

getOptions(options: OptionsSelect): Promise<boolean>

Populates select/dropdown elements with options fetched from a URL

Parameters:
options OptionsSelect mandatory

Configuration object containing:

  • url: API endpoint to fetch options data from
  • dropDowns: Array of jQuery select elements to populate
  • firstText: Text to show as first/placeholder option
  • text: Property name to use as option text
  • internal: Property name to use as option value
  • jwt: JWT token for authentication
  • lang: Language/culture code (default: 'en-US')

Examples

Basic AJAX Request

// GET request with JWT authentication
SweetMeSoft.get({
    url: '/api/users',
    jwt: 'your-jwt-token',
    lang: 'en-US',
    successCallback: (data) => {
        console.log('Users:', data);
    }
});

// POST request
SweetMeSoft.post({
    url: '/api/users',
    data: { name: 'John Doe', email: 'john@example.com' },
    jwt: 'your-jwt-token',
    successCallback: (response) => {
        console.log('User created:', response);
    }
});

Populate Select Elements

SweetMeSoft.getOptions({
    url: '/api/countries',
    dropDowns: [$('#countrySelect')],
    text: 'name',
    internal: 'id',
    firstText: 'Select a country',
    jwt: 'your-jwt-token'
});

Generate Chart

SweetMeSoft.generateChart({
    chart: $('#salesChart'),
    url: '/api/sales-data',
    interval: 'monthly'
});

jQuery Extensions

// Initialize Bootstrap Select
$('#mySelect').initializeSelect();

// Check if string is empty
if (userInput.isNullOrEmpty()) {
    alert('Please enter a value');
}

// Enable/disable elements
$('#submitButton').disable();
$('#submitButton').enable();
